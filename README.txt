Software to build and maintain SCOPe, https://scop.berkeley.edu/

This software has dependencies, including a local MySQL database
with all the data (see downloads below), and StrBioLib
(see https://www.ncbi.nlm.nih.gov/pmc/articles/PMC4566930/
 and https://sourceforge.net/projects/strbio/)
